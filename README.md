# 🐈‍⬛ Cyber Catppuccin for Blender

A sleek, high-contrast **Catppuccin Mocha** theme for Blender, infused with a vibrant **Cyberpunk** aesthetic. Designed for maximum readability and a tactile, high-tech feel.

![Cyber Catppuccin Banner](https://raw.githubusercontent.com/Demigirlboss0/blender-cyber-catppuccin/main/Banner.svg)

## ✨ Features

- **Catppuccin Mocha Palette**: Uses the official `Crust`, `Mantle`, `Base`, and `Mauve` colors for a soothing yet professional dark look.
- **Cyberpunk Accents**: High-contrast `Sapphire` and `Mauve` highlights for a modern HUD experience.
- **Tactile Feedback**:
  - **Hover**: Interactive elements glow **Sapphire Blue** on hover.
  - **Active**: Checked boxes and active toggles glow **Mauve Purple**.
  - **Sliders**: Unique interaction with Sapphire borders/text and a deep, saturated Mauve fill bar.
- **Optimized for Blender 4.2+**: Fully compatible with the new Blender Extensions system.
- **Maximized Readability**: Fine-tuned contrast for node editors, popup dialogues, and translucent windows.

## 🚀 Installation

### As an Extension (Blender 4.2+)

1. Download the `theme-cyber-catppuccin.zip` from the [Releases](https://github.com/Demigirlboss0/blender-cyber-catppuccin/releases) page.
2. Open Blender and go to **Edit > Preferences > Get Extensions**.
3. Click the arrow icon (Install from Disk) in the top-right corner.
4. Select the `.zip` file and install.

### Manual Installation (Legacy)

1. Download the `CyberCatppuccin.xml` file.
2. In Blender, go to **Edit > Preferences > Themes**.
3. Click **Install...** at the bottom and select the `.xml` file.
