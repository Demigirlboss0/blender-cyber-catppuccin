# 🐈‍⬛ Cyber Catppuccin for Blender

A sleek collection of high-contrast **Catppuccin** themes for Blender, infused with a vibrant **Cyberpunk** aesthetic. Choose your roast: the deep, digital void of **Mocha** or the warm, creamy energy of **Latte**.

![Cyber Catppuccin Banner](https://raw.githubusercontent.com/Demigirlboss0/blender-cyber-catppuccin/main/Banner.svg)

## ✨ Variants

### ☕ Cyber-Catppuccin Mocha (Dark)
- **Palette**: `Crust`, `Mantle`, and `Base` for an "infinite void" HUD feel.
- **Accents**: Vibrant **Mauve** and **Sapphire** highlights.
- **Tactile Feedback**: Interactive elements glow **Blue (Sapphire)** on hover and **Purple (Mauve)** when active.

### 🥛 Cyber-Catppuccin Latte (Warm Light)
- **Palette**: Warm **Cream** and **Steamed Milk** beige tones—designed to be easy on the eyes.
- **Text**: Rich **Espresso brown** for soft, high-readability contrast.
- **Tactile Feedback**: Clean, monochromatic **Lavender** glow for both hover and active states.

## 🚀 Installation

### As an Extension (Blender 4.2+)

1. Download the preferred `.zip` variant (`mocha` or `latte`) from the [Releases](https://github.com/Demigirlboss0/blender-cyber-catppuccin/releases) page.
2. Open Blender and go to **Edit > Preferences > Get Extensions**.
3. Click the arrow icon (**Install from Disk**) in the top-right corner.
4. Select the `.zip` file and install.

### Manual Installation (Legacy)

1. Download the relevant `.xml` file from the repository.
2. In Blender, go to **Edit > Preferences > Themes**.
3. Click **Install...** at the bottom and select the `.xml` file.
